Bot commands:

!stayout.bot -new: gives a brief summary of what has happened on Hive in the last 24 hours (how many posts, comments, upvotes, and which users wrote their very first post)

!stayout.bot -dv: how many downvotes you have ever received and given, and from whom and to whom the most

!stayout.bot -stats: how many posts and comments you have written in total, average payout, how often per week and on which day you have written the most

plus, it automatically checks the chain to see if an account is posting for the very first time, then sends it a short welcome message

Don´t forget to customize before using (bot name, posting key, welcome message text)
